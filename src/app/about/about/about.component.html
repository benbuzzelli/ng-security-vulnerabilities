Paul Gazzilo
Sponsor

Our sponsor Paul Gazzilo came to us with a vision of an open source project that will be able to sift through your code in search of security vulnerabilities. By using github repositories, you will be able to link your github and us our machine learning model to see just how secure your code really is.
Security Vulnerability
Main Reason

When thinking about security and how much it is needed, you should also think just how secure the code you write really is. Our senior design project will be able to help you put your code to the test. While this is a good starting point, you will be able to add code to our open source database.
Senior Design
Our Mission

With our sponsor, we are working together to make a public web app that will be able to introduce security vulnerabilities to any projects. Our mission with seeks to create a reliable place to test for security vulnerabilities. With this idea we want to be able to further the reach of security assessment for anyone with a github link.
